# popular-quotes
A popular quotes package to get you quotes from various sources of media


# Data Structure
`tv.json`
```
	{
		"quote": "TEST",
		"character": "",
		"title": "",
		"season": 1,
		"episode": 1,
		"year": 2001
	}
	{
		"quote": “Most people are nothing but pawns on a chessboard led by an unknown hand”,
		"character": "Noah",
		"title": "Dark",
		"season": 1,
		"episode": 10,
		"year": 2017
	}
	
	
```

`movies.json`
```
	{
		"quote": "TE32312ST2",
		"character": "",
		"title": "",
		"actor": "",
		"year": 2001
	}
```

`anime.json`
```
	{
		"quote": "TEST4342342",
		"episode": 123,
		"title": "",
		"year": 2001
	}
```

`books.json`
```
	{
		"quote": "TES312312T2",
		"character": "",
		"title": "",
		"author": "",
		"year": 2001
	}
```

`speeches.json`
```
	{
		"quote": "T32312",
		"person": "",
		"year": 2001
	}
```
